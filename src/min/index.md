title: min
---

> Computes the minimum value of array. If array is empty or falsey, undefined is returned.

## Arguments

- array (Array): The array to iterate over.

## Returns

- (\*): Returns the minimum value.

## Example

```ts
min([4, 2, 8, 6])
// => 2

min([])
// => undefined
```
