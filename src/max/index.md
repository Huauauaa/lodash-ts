title: max

> Computes the maximum value of array. If array is empty or falsey, undefined is returned.

## Arguments

- array (Array): The array to iterate over.

## Returns

- (\*): Returns the maximum value.

## Example

```ts
max([4, 2, 8, 6])
// => 8

max([])
// => undefined
```
